# PhoneBid Client

The frontend application for PhoneBid - the world's first anonymous phone auction marketplace.

## Features

- **Anonymous Trading**: Complete privacy protection for all users
- **Secure Auctions**: Real-time bidding with escrow payments
- **Modern UI**: Built with React, Tailwind CSS, and Lucide React icons
- **Responsive Design**: Optimized for all devices
- **SEO Optimized**: Comprehensive meta tags and structured data

## Tech Stack

- **React 18** - Modern React with hooks
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful, customizable icons
- **React Router** - Client-side routing
- **Axios** - HTTP client for API calls
- **React Hot Toast** - Elegant notifications

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Logo & Branding

The PhoneBid logo uses a modern smartphone icon from Lucide React, enclosed in a bright green (#c4ff0d) rounded container. The logo is implemented as a reusable component (`Logo.jsx`) with different size variants:

- `sm` - Small size for compact spaces
- `md` - Medium size for navbar (default)
- `lg` - Large size for footer and headers
- `xl` - Extra large for hero sections

## SEO Features

The application includes comprehensive SEO optimization:
- Meta descriptions and keywords
- Open Graph tags for social sharing
- Twitter Card support
- Structured data (JSON-LD)
- Canonical URLs
- Custom favicon and app icons
